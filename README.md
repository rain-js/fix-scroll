# fix-scroll

> Vue directive to fix layer scroll

## Installation

``` bash
$ npm i fix-layer-scroll
```

## Usage

main.js 
``` bash
import fixScroll from 'fix-layer-scroll'

Vue.use(fixScroll)
```

template
``` bash
<div class="dialog-mask" v-fixscroll>
    <div class class="dialog-wrapper">
    </div>
</div>
```
